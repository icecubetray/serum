◆ lssig_hash_finish

typedef ls_result_t(* lssig_hash_finish) (void *const restrict context, const uint8_t *const restrict data, const size_t size, uint8_t *const restrict digest)

Calculates the digest for the specified context, clearing the context afterwards.

Transforms the remaining input, if any, and finishes the message with proper padding. Calculates the digest, and clears the context.

Parameters
contextThe context to use.
dataRemaining input to transform. May be NULL if there is nothing to transform (however, size must be set to 0).
sizeThe size of the remaining input.
digestThe output location of the digest.
Returns
LS_E_NULL if context or digest is NULL.
LS_E_NULL if size is greater than 0 and input is NULL.
LS_E_SIZE if size is greater than the algorithm's block size.
LS_E_FAILURE if any transformations fail.
LS_E_SUCCESS otherwise.
