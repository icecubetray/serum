ls_result_t ls_memory_dump_ex ( const void *const  src,
const size_t  size,
const unsigned int  columns,
const unsigned int  columnsize,
FILE *const  dest,
const ls_bool_t  ansi 
)

Dumps the memory in the range from src to src + size.

Writes the dump to the file specified by dest.

Parameters
srcThe starting address to dump from.
sizeThe number of bytes to dump.
columnsThe number of columns to display.
columnsizeThe number of bytes per column.
destA pointer to the FILE to dump to.
ansiWhether or not to enhance the output using ANSI escape codes.
Returns
LS_E_NULL if src or dest is NULL.
LS_E_NOOP if size is less than 1.
LS_E_INVALID if columns or columnsize is less than 1.
LS_E_IO_WRITE if writing to dest fails.
LS_E_IO_FLUSH if flushing to dest fails.
LS_E_SUCCESS otherwise.

◆ ls_memory_to_c_array()

const size_t  size 
)

Converts the memory in the range from src to src + size to a string representing the memory as a C array of bytes.

Each successful call to this function must be countered by a call to free once, to deallocate the allocated memory.

Parameters
srcThe starting address to dump from.
sizeThe number of bytes to dump.
Returns
NULL on failure.
A pointer to the newly allocated string otherwise.

◆ ls_memory_free()

void* ls_memory_free ( void *const  target)

Wrapper for the standard free function.

Unlike standard free, this function always returns NULL and checks if the passed pointer isn't null before passing it down.

Parameters
targetThe pointer to the memory to deallocate.
Returns
NULL, always.
