Finishes a hash context with 0x80,{0x00},<u64mesglen>.

Pads the remaining data with one bit (0x80), filling the remainder of the buffer to the last 8 bytes with zeroes, and adding the total length of the message in those 8 bytes as a 64-bit little-endian integer.

Parameters
[in,out]contextThe hash context to finish.
[in]f_updateThe hash update function to use.
[in]lengthThe length of the message.
[in,out]block_bufferThe block to use as a buffer.
[in]block_sizeThe size of the block.
[in]inputThe remaining input to process.
[in]input_sizeThe size of the remaining input.
Returns
LS_E_NULL if context, f_update, or block_buffer, or, if input_size is greater than 0, input is NULL.
LS_E_SIZE if block_size is 0.
LS_E_FAILURE if f_update doesn't return LS_E_SUCCESS.
LS_E_SUCCESS otherwise.`
